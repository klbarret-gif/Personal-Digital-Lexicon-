# Personal-Digital-Lexicon-
A collection of my digital language to friends

What I say and how I say it, especially without a filter, reflects who I am.

##Process:

My best friend from high school and I still use Discord to talk because of the years of messages we have on there. In order to read and analyze these messages, I used Discrub, an incredibly useful chrome extension. Unfortunately, it takes hours to compile incredibly large amounts of text in Discord.

From 2023 to now, I have sent over 150k messages to my best friend and counting. This data dictated most of my results.

I also included some messages sent in recent groupchats made with my friend group when adding to emoji's or certain phrases frequencies.

Discrub saves message content into a csv file, which I read and analyzed in Python to find the phrases I use the most. I used code from Stack Overflow and Google with some slight adjustments. Then I searched for these words, phrases, and emoji's in Discord, screen-shotting ones I found funny, but not too personally incriminating.
